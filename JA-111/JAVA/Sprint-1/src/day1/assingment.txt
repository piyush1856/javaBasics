What are the benefits of using Java programming language?
- Java is easy to learn and its syntax is simple, clean
- it supports the oops characters,it support the oops characters
- java is platform independent
- it has security against malicious code
- java is robust language, robust means strong

Why is Java known as a platform independent language?
- Java is platform independent language because we can code in different operating system and run it on different OS
- source code is complied and converted into bytecode and this byte code run into different platform
- WORA - Write Once Run Anywhere
